# Jetpack_Joyride_in_Rust_with_Bevy

Я зробив гру Jetpack Joyride на Rust з Bevy.

## Про Jetpack Joyride
Jetpack Joyride — відеогра жанру сайд-скролер, нескінченний ранер/екшн, створена Halfbrick Studios. Спочатку випущена для IOS пристроїв на App Store 1 вересня 2011 року[1], а згодом гра була портована і на безліч інших систем. Була випущена в Інтернеті, як флеш-версія 11 травня 2012 року[2], на Android — 28 вересня[3], на PlayStation Portable (через PlayStation Network, портована на Beatshapers) — 20 листопада, у Північній Америці та в Європі — 21 листопада[4]; на BlackBerry PlayBook — 13 грудня 2012[5], на PlayStation 3 і PlayStation Vita (через PlayStation Network, портована на Big Ant Studios) — 21 грудня в Європі та 31 грудня у Північній Америці[6]; на BlackBerry 10 — 6 березня 2013[6], а на Windows Phone 8 і Windows 8 — 5 червня[7][8].
Гра має такого ж героя, як і у Monster Dash, Баррі Стейкфрайз (англ. Barry Steakfries), який краде реактивний ранець з надсекретної лабораторії, яким і управляє гравець. Гра отримала дуже схвальні відгуки, і завоювала безліч нагород.
Баррі Стейкфрайз працює продавцем у комппанії, що виготовляє грамофони, але бізнес невдовзі збанкрутує через падіння продажів. Одного разу, коли він, засмучений, ходив по вулиці, знаходить одну з «надсекретних» лабораторій «Legitimate Research» (укр. Законні дослідження), і бачить реактивний ранець, що оснащений кулеметами. Мріючи про використання реактивного ранця задля добрих справ, Баррі вдирається через стіни до лабораторії та краде експериментальний реактивний ранець з лап науки.
## Про Rust
Rust (укр. Раст, досл. «Іржа») — сучасна багатопарадигмова мова програмування загального призначення. Мова має сильну (строгу) типізацію і зосереджена на безпечній роботі з пам'яттю та забезпеченні високої рівночасності виконання задач (можливість породжувати тисячі й навіть мільйони підпроцесів).
Початковий код проєкту поширюється під ліцензією MIT.
За структурою, мова Rust нагадує C++, але істотно відрізняється в деяких дрібницях реалізації синтаксису та семантики, а також зосередженням на блокову організацію структури коду, яка дає змогу реалізувати завдання, подібно до легковагих співпрограм. Автоматичне керування пам'яттю позбавляє розробника потреби маніпулювання вказівниками й захищає від проблем, що виникають через низькорівневу роботу з пам'яттю, таких як звернення до ділянки пам'яті після її звільнення, розіменовування нульових вказівників, вихід за межі буфера тощо. Rust підтримує суміш імперативних, процедурних і об'єктно-орієнтованих методів з такими парадигмами, як функційне програмування і модель акторів, а також узагальнене програмування і метапрограмування, у статичних і динамічних стилях.
Rust — це високоефективна, безпечна для пам'яті мова програмування, яка активно використовується у системному програмуванні, розробці високонавантажених серверів, блокчейн-проектах, вбудованих системах (IoT) та WebAssembly. Вона забезпечує швидкість C++ без ризиків безпеки завдяки унікальній моделі управління пам'яттю.
Офіційним та найкращим місцем для завантаження мови програмування Rust є сайт rust-lang.org. Встановлення відбувається через інструмент rustup, який керує версіями Rust.
Як встановити:
-Windows, macOS, Linux: Перейдіть на сторінку install та дотримуйтесь інструкцій для вашої ОС.
-Термінал (macOS/Linux): Найшвидший спосіб — запустити команду:
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh.
-Windows: Завантажте та запустіть rustup-init.exe. Може знадобитися встановлення Visual Studio C++ Build tools.
Після встановлення ви отримаєте компілятор rustc, менеджер пакетів cargo та документацію.
## Про Bevy
Bevy — це сучасний, високопродуктивний ігровий рушій з відкритим кодом, написаний мовою програмування Rust. Він використовує архітектуру ECS (Entity Component System), що робить його чудовим для 2D та 3D ігор, особливо при роботі з великою кількістю об'єктів. Bevy вирізняється модульністю, швидкістю та є перспективним вибором для розробників, які цінують продуктивність.
Основні характеристики та особливості Bevy:
-Мова Rust: Забезпечує високу безпеку пам'яті та швидкість виконання.
-Архітектура ECS: Entity Component System дозволяє ефективно організувати дані та логіку ігор.
-Відкритий код: Розвивається спільнотою, підтримує 2D та 3D.
-Модульність: Можливість підключати лише необхідні плагіни, що робить його гнучким.
-Продуктивність: Добре підходить для складних систем з великою кількістю акторів. 
Для кого підходить:
Bevy є чудовим вибором для тих, хто хоче створювати продуктивні ігри на Rust, але через його розвиток може потребувати вивчення нових концепцій.

-GitHub: {Link: github.com/bevyengine/bevy https://bevy.org/} — для завантаження сирцевого коду.
-Cargo (Rust): Додається до Rust-проекту через файл Cargo.toml.
